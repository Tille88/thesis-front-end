Instructions

This is data gathering for a M.Sc.-study evaluating the usefulness of different legend-types in data-visualizations using maps.

You will be be presented with 10 randomly generated sample images.

In this images you will be asked to report the data-value you guess is represented by the overlay-opacity layer at the location of a marker

Elements

Base Map

The base map is similar to a map you use every day online or on Google maps.

Data Layer

A data variable is represented using opacity of different colours.
The more opaque (darker) the data layer is - the higher the value.

Legends

A number of different legend types will be shown - one for each image (except first and last image which are used as baseline).
Please use the legend of the image in order to estimate the value at the location of the the marker

Marker

A marker indicates the location for you to guess the value of the data layer. The location of the marker is random within the extent of the data-layer.

Input Slider

You input your guess using the slider. The starting position is generated randomly.
Before any change is made, it is not possible to move to the next picture.

Next/Submit

And when you are ready, you click through to the next image.
