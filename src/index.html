Transparency Data Mapping

Approximate time required 10 minutes

Instructions

This is data gathering for a study evaluating the perception-accuracy of opacity/transparency in the context of static 2D-maps.

You will be be presented with 8 different sample images. In this images you will be asked to report the data-value you believe is represented by the overlay-opacity layer at the location of a marker

Elements

Base Map

The base map is similar to a map you use every day online or on Google maps.

Data Layer

A data variable is represented using opacity of different colours.

Legends

A number of different legend types will be shown - one for each image.

Marker

A marker indicates the location for you to guess the value of the data layer.

Input Slider

You input your guess using the slider

Next/Submit

And when you are ready, you click through to the next image.

Progress Indicator

A progress indicator guides you through the progress.

Window size

It is strongly recommended to use a desktop browser - as the images are static. But if you're on a phone you can try to zoom into the pictures.
